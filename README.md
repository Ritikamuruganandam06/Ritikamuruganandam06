## Featured Projects

### AI-Powered Database Assistant
`FastAPI` `React` `LangChain` `PostgreSQL` `GPT` `Gemini` `Claude`

A web-based AI assistant that lets you talk to your database in plain English. Type a question, get SQL — no query writing needed. Built a chat interface that generates and executes SQL queries, visualizes schemas dynamically, and routes requests through multiple LLM providers (Azure OpenAI, Gemini, OpenAI, Anthropic) via LangChain for intelligent, context-aware query processing.


### Personal Blog Site
`React` `Vite` `Markdown`

A blog integrated directly into my portfolio, powered by a file-based content pipeline I built from scratch. Posts are written in Markdown with YAML frontmatter, and a custom Vite plugin extracts metadata (title, tags, date, reading time) at build time through a virtual module — no CMS, no database, just files. Includes paginated listing, tag filtering, prev/next navigation, and a dark/light theme toggle.


### Web-Based IoT Dashboard for Real-Time Sensor Monitoring
`React.js` `Node.js` `Express.js` `MySQL` `Prisma` `Socket.IO`

A full-stack IoT dashboard built to visualize live sensor data across multiple devices simultaneously. RESTful APIs handle device management and structured data via MySQL with Prisma ORM, while Socket.IO keeps the frontend in sync with real-time bi-directional events. Designed to scale cleanly as device count grows.

> 🥉 2nd Runner-up — CDAC Urban IoT Challenge '25


### Food Waste Management & Donation Platform
`Next.js` `Node.js` `Express.js` `MongoDB`

A role-based web application that streamlines the full lifecycle of food donations — from listing surplus food to volunteer assignment and delivery confirmation. Implements JWT authentication, role-based authorization, and admin-controlled NGO verification. Donation state is tracked end-to-end: request → approval → volunteer assignment → status updates.
